# 一些说明
❗ 本文中没有任何aff链接 没有获利 也不对任何机场的行为负责
```
aff:
    Affiliate,简单来说就是推广
    aff链接的末尾通常会跟随一段代表邀请人id的邀请码 如:https://www.baidu.com/?aff=0721
    通过链接注册消费后,邀请人会获得一定比例的返利  
    因此会有很多人自发的去推广一些机场 这些人被称为affman

流媒体解锁:
    解除地域限制,可以观看NETFLIX、HULU、HBO、AbemaTV、Disney等流媒体内容

倍率:
    有些机场不同的节点拥有不同的倍率 倍率越低消耗流量越少
    比如使用0.1倍率的节点跑10G流量,实际上只消耗账户中的1G流量
    如果使用2倍率的节点跑10G流量则会消耗账户中的20G流量

同时在线设备数量:
    注意是在线设备 即正在使用节点加速的设备
    通常机场会对不同的订阅进行设备数量限制 超出限制后可能无法使用
    情节严重的有被ban账户的可能性
```
